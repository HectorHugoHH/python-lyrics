# python-lyrics / en

A program made to display song lyrics line by line on popups using tkinter on python (First project, inspired by tiktok popup lyrics videos)

Solves the problem of having to manually enter and separate the song lyrics on a list, done by automatically separating each newline in a different element of a list using the loop 'for' and file management to display every line on a new message box when clicking Ok.




# python-lyrics / es

Programa hecho para mostrar letras de canciones por verso en popups usando tkinter en python (Primer proyecto, inspirado en videos de tiktok de letras de canciones en popups)

Resuelve el problema de tener que escribir y separar la letra de las canciones en una lista manualmente, separando automaticamente cada linea en un elemento distinto de una lista usando el loop 'for' y gestión de archivos para mostrar cada verso en un nuevo popup cada vez que presionas Aceptar.
